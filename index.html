Tram Times

Usually in order to find out when the next tram is due, you need to go to the actual tram stop in order to see times board. This site allows you to search up the tram stop to view the next 3 tram times and the messages displayed on the board.


It also provides the weather for the destination with a background matching the weather forecast shown.


Built Using:

  • HTML
  • CSS & CSS Animations
  • JavaScript
  • Metrolink API
  • OpenWeather API
    •
  • Version Control through Git & GitHub

News Aggregator

A site that aggregates news stories from multiple news sites to keep you up to date and provides filters for specific filter types to display only news articles of that genre.


The article images update per article displaying relevant images to the article. Article size it set to random allowing for a variation on the view.


Due to the API license used, this will only work when deploying to local host.


Built Using:

  • HTML
  • CSS & CSS Animations
  • JavaScript
  • React & React States
  • News API
    •
  • Version Control through Git & GitHub

Social Media Site

A Twitter like social media site built with a React Front End and a nodeJS Firebase back end.


Allowing for creating new posts, liking, commenting, allowing for notifications to be received, profile picture uploading and profile creation. Users can delete their own posts

  • HTML
  • CSS
  • JavaScript

Personal Dashboard

A dashboard that provides weather for the next 12 hours, your Twitter feed including images, and a drag & drop organisable to do list utilising local storage in browser to remember the content & order of the content.


The weather section updates the images & background dynamically to show the current weather and the forecast.


Built Using:

  • HTML
  • CSS
  • JavaScript
  • React & React States
  • NodeJS Server
  • Twitter API
    •
  • OpenWeather API
    •
  • OAuth Server Deployed to Heroku
  • Version Control through Git & GitHub
